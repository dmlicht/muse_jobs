# Download Jobs
Download jobs from The Muse's Jobs Api

## Usage
sort a directory:

    python muse download --pages 10 # download 10 pages of jobs from the muse api
    python muse show --location "New York City Metro Area"

Get help:

    python muse -h

## Installation

From the root directory:

    pip install -e .
